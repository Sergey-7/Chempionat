# Задача «Разработка модели для сегментации объектов на планах помещений по изображениям»

В рамках проекта Цифровой прорыв, нацеленным на реализацию применения ТИМ (технологий информационного моделирования) в процессе строительства, реконструкции и эксплуатации объектов, финансируемых за счет средств государственного бюджета, первым шагом на пути стандартизации и цифровизации государственной и муниципальной недвижимости является сегментация основных объектов помещения, таких как стены, окна, оборудование, двери.
В рамках задачи регионального чемпионата проекта Цифровой прорыв для автоматического распознавания планов участники чемпионата занимались разработкой моделей семантической сегментации, принимающих на вход растровое изображение, содержащее план жилого помещения.
Использовался датасет, разработанный методом кроссвалидационной ручной разметки, для обучения модели, осуществляющей стандартизацию формата начертания плана.
